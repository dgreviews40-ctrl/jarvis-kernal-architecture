# __PLUGIN_NAME__

A JARVIS UI panel plugin built with React.

## Features

- React-based UI components
- Access to JARVIS memory
- Configurable themes
- Real-time memory search

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Customization

Edit `src/Panel.tsx` to customize the UI. Available hooks:

- `usePluginContext()` - Access plugin context
- `usePluginMemory()` - Store/recall memories
- `usePluginConfig()` - Read/write configuration
- `useNotification()` - Show notifications

## Building

```bash
npm run build
```
